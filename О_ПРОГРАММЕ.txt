СОСТАВ ПРОЕКТА:
---------------

📄 analyzer_comparison.py
   Основная программа с графическим интерфейсом
   Функционал:
   - Загрузка Excel файлов с данными измерений
   - Построение интерактивных графиков
   - Перекрестие для точного считывания значений
   - Автоматическое определение структуры данных

📄 create_test_data.py
   Скрипт для создания тестовых данных
   Генерирует примерные файлы Excel для демонстрации работы

📄 requirements.txt
   Список необходимых библиотек Python

📄 run.bat
   Скрипт быстрого запуска программы для Windows

📄 create_test.bat
   Скрипт для создания тестовых данных

📄 README.md
   Техническая документация (на английском)

📄 ИНСТРУКЦИЯ.txt
   Подробная инструкция по использованию (на русском)

📄 О_ПРОГРАММЕ.txt
   Этот файл - общее описание проекта


БЫСТРЫЙ СТАРТ:
--------------

1. Установите зависимости:
   
   Откройте командную строку в папке с программой и выполните:
   > pip install -r requirements.txt

2. Запустите программу:
   
   Способ А: Двойной клик по файлу run.bat
   Способ Б: В командной строке выполните:
   > python analyzer_comparison.py

3. Загрузите ваши файлы данных:
   
   - Нажмите "📁 Загрузить файл H2S"
   - Нажмите "📁 Загрузить файл SO2"
   - Нажмите "📊 Построить графики"

4. Работа с графиками:
   
   - Наведите курсор на график для просмотра значений
   - Используйте колесико мыши для масштабирования
   - Перетаскивайте график для панорамирования


СОЗДАНИЕ ТЕСТОВЫХ ДАННЫХ:
--------------------------

Если у вас еще нет данных для тестирования:

1. Выполните команду:
   > python create_test_data.py

2. Будут созданы файлы:
   - test_H2S_data.xlsx (360 записей, 2 недели данных)
   - test_SO2_data.xlsx (360 записей, 2 недели данных)

3. Загрузите эти файлы в программу для тестирования

ТЕХНИЧЕСКИЕ ХАРАКТЕРИСТИКИ:
----------------------------

Язык программирования: Python 3.7+

Используемые библиотеки:
- pandas 1.3.0+ (работа с данными)
- numpy 1.21.0+ (численные операции)
- PyQt5 5.15.0+ (GUI)
- pyqtgraph 0.12.0+ (графики)
- openpyxl 3.0.0+ (Excel)

Размер программы: ~20 КБ (без библиотек)
Потребление памяти: ~50-100 МБ (зависит от объема данных)
Поддерживаемые ОС: Windows 7/10/11


СТРУКТУРА ДАННЫХ:
------------------

Программа работает с Excel файлами следующей структуры:

┌──────────────────────┬───────────────┬───────────────┬─────────┐
│ Колонка 1            │ Колонка 2     │ Колонка 3     │ ...     │
│ (Время/Дата)         │ (Параметр 1)  │ (Параметр 2)  │         │
├──────────────────────┼───────────────┼───────────────┼─────────┤
│ 2024-10-13 00:00:00  │ 5.234         │ 10.456        │ ...     │
│ 2024-10-13 01:00:00  │ 5.123         │ 10.345        │ ...     │
│ 2024-10-13 02:00:00  │ 5.345         │ 10.567        │ ...     │
│ ...                  │ ...           │ ...           │ ...     │
└──────────────────────┴───────────────┴───────────────┴─────────┘

Требования:
- Первая колонка или колонка с названием, содержащим "время",
  "дата", "time", "date" используется как временная ось
- Остальные колонки должны содержать числовые данные
- Рекомендуемый формат даты: YYYY-MM-DD HH:MM:SS


ПРИМЕРЫ ИСПОЛЬЗОВАНИЯ:
-----------------------

ПРИМЕР 1: Сравнение двух анализаторов H2S
   1. Загрузите файл с данными первого анализатора как "H2S"
   2. График покажет все параметры из файла
   3. Используйте перекрестие для анализа конкретных моментов

ПРИМЕР 2: Сравнение H2S и SO2 одновременно
   1. Загрузите файл H2S
   2. Загрузите файл SO2
   3. Оба графика отобразятся друг под другом
   4. Перекрестие работает независимо на каждом графике

ПРИМЕР 3: Анализ расхождений
   1. Загрузите файлы с данными обоих анализаторов
   2. Наведите курсор на интересующую точку
   3. Сравните значения в информационной панели
   4. Масштабируйте для детального просмотра
