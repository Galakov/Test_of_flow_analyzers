  ПРОГРАММА ДЛЯ СРАВНЕНИЯ АНАЛИЗАТОРОВ SO2 и H2S
═══════════════════════════════════════════════════════════════════

ОПИСАНИЕ:
---------
Программа предназначена для визуального сравнения данных с двух
анализаторов путем построения временных рядов с интерактивным 
перекрестием для точного определения значений в любой момент времени.

УСТАНОВКА:
----------
1. Убедитесь, что у вас установлен Python 3.7 или выше
2. Откройте командную строку в папке с программой
3. Выполните команду:
   pip install -r requirements.txt

ЗАПУСК ПРОГРАММЫ:
-----------------
Способ 1: Двойной клик по файлу run.bat
Способ 2: В командной строке выполните:
          python analyzer_comparison.py

ИСПОЛЬЗОВАНИЕ:
--------------
1. После запуска программы откроется главное окно

2. Нажмите кнопку "📁 Загрузить файл H2S" и выберите Excel файл
   с данными измерений H2S

3. Нажмите кнопку "📁 Загрузить файл SO2" и выберите Excel файл
   с данными измерений SO2

4. Нажмите кнопку "📊 Построить графики"

5. Графики отобразятся в окне программы

6. Для просмотра точных значений:
   - Наведите курсор мыши на любую точку графика
   - В верхней части окна отобразится информация:
     * Дата и время измерения
     * Значения всех параметров в этот момент
   - Перекрестие покажет точное положение на графике

7. Функции управления графиками:
   - Масштабирование: колесико мыши или перетаскивание с зажатой
     правой кнопкой мыши
   - Панорамирование: перетаскивание с зажатой левой кнопкой мыши
   - Сброс масштаба: правый клик → "View All"

8. Кнопка "🗑️ Очистить" - удаляет все загруженные данные и графики

ФОРМАТ ВХОДНЫХ ДАННЫХ:
-----------------------
Программа автоматически распознает структуру Excel файлов:

- Первая колонка или колонка с названием, содержащим слова
  "время", "дата", "time", "date" используется как временная ось

- Остальные числовые колонки отображаются как параметры для
  сравнения

Пример структуры файла:
┌─────────────────────┬──────────────────┬──────────────────┐
│ Дата и время        │ Анализатор 1     │ Анализатор 2     │
├─────────────────────┼──────────────────┼──────────────────┤
│ 2024-10-13 00:00:00 │ 5.234            │ 5.456            │
│ 2024-10-13 01:00:00 │ 5.123            │ 5.345            │
│ ...                 │ ...              │ ...              │
└─────────────────────┴──────────────────┴──────────────────┘

СОЗДАНИЕ ТЕСТОВЫХ ДАННЫХ:
--------------------------
Если вы хотите протестировать программу с примерными данными:

1. Выполните команду:
   python create_test_data.py

2. Будут созданы два файла:
   - test_H2S_data.xlsx
   - test_SO2_data.xlsx

3. Используйте эти файлы для тестирования программы

ВОЗМОЖНОСТИ:
------------
✓ Загрузка Excel файлов (.xlsx, .xls)
✓ Автоматическое определение структуры данных
✓ Одновременное отображение нескольких параметров
✓ Интерактивное перекрестие с отображением значений
✓ Масштабирование и панорамирование графиков
✓ Сравнение данных с разных анализаторов
✓ Отображение даты и времени в читаемом формате

СИСТЕМНЫЕ ТРЕБОВАНИЯ:
---------------------
- Windows 7/10/11
- Python 3.7 или выше
- 2 ГБ свободной оперативной памяти
- 100 МБ свободного места на диске

ЗАВИСИМОСТИ:
------------
- pandas (работа с данными)
- numpy (численные вычисления)
- PyQt5 (графический интерфейс)
- pyqtgraph (построение графиков)
- openpyxl (чтение Excel файлов)

РЕШЕНИЕ ПРОБЛЕМ:
----------------
Проблема: Программа не запускается
Решение: Убедитесь, что установлены все зависимости:
         pip install -r requirements.txt

Проблема: Файл не загружается
Решение: Проверьте формат файла (должен быть .xlsx или .xls)
         Убедитесь, что файл не поврежден

Проблема: График не отображается
Решение: Проверьте, что в файле есть числовые данные
         Убедитесь, что есть колонка с датой/временем

Проблема: Перекрестие не работает
Решение: Убедитесь, что курсор находится над областью графика
